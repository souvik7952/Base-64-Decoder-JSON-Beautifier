<h2>About This Project</h2>

<p>
This is a lightweight, fully client-side <b>Base64 Decoder Tool</b> built using only HTML, CSS, and JavaScript within a single file.
It allows users to easily decode Base64-encoded data into readable text with a clean and modern interface.
</p>

<h3>Key Features</h3>
<ul>
  <li>Paste or upload Base64 encoded data</li>
  <li>Instant decoding with error handling</li>
  <li>Beautify decoded output into formatted JSON</li>
  <li>Download decoded text as <code>.txt</code> file</li>
  <li>Download formatted JSON as <code>.json</code> file</li>
  <li>Copy to clipboard and clear functionality</li>
  <li>Responsive and user-friendly UI</li>
</ul>

<h3>Tech Stack</h3>
<p>
Pure HTML with embedded CSS and JavaScript (No external libraries or frameworks).
</p>

<h3>Usage</h3>
<p>
Simply open the HTML file in any modern browser, paste or upload your Base64 data, and start decoding instantly.
</p>

<h3>Note</h3>
<p>
This tool works entirely in the browser, ensuring privacy and no data is sent to any server.
</p>
